# 🚀 Local AI IDE - Desktop Pro

The easiest way to use this application is to download the pre-built installer. **No coding knowledge or terminal commands required.**

## 📥 One-Click Installation

1. Go to the [**Releases**](https://github.com/YOUR_USERNAME/YOUR_REPO/releases) page of this project.
2. Download the latest `Local-AI-IDE-Setup.exe`.
3. Run the installer and start coding instantly.

---

## 🛠 Features for Desktop Engineering
- **Pure Offline Mode**: All AI data and code stays on your disk.
- **Embedded Core Engine**: Integrated Python backend and Node-PTY terminal.
- **Auto-Sync**: Automatically detects local Ollama models.

## 🏗 Developer: How to build the .exe manually
If you want to build the installer yourself:
1. `npm install`
2. `npm run package`
The installer will be generated in the `/release` folder.
